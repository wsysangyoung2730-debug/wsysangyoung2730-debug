- Implemented CSV export for machine-learning datasets.

### CultureKeeper

| | |
| --- | --- |
| **Period** | July 2026 |
| **Summary** | CultureKeeper is a culture-sharing platform that reinterprets vacant and aging spaces in Suseong-gu, Daegu, as shared cultural hubs for young creators and local residents. |
| **Role** | Service planning · Business model design · User-flow design · Web prototype development and deployment |
| **Result** | HYPECRAFT BEST INNOVATION AWARD · Best Innovation Award at the WCIF HYPECRAFT Culture Hackathon, Track B |
| **Tech Stack** | ![React](https://img.shields.io/badge/React-20232A?style=flat-square&logo=react&logoColor=61DAFB) ![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=flat-square&logo=typescript&logoColor=white) ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat-square&logo=javascript&logoColor=222222) ![Kakao Maps JavaScript API](https://img.shields.io/badge/Kakao%20Maps%20JavaScript%20API-FFCD00?style=flat-square&logo=kakao&logoColor=222222) ![Vite](https://img.shields.io/badge/Vite-646CFF?style=flat-square&logo=vite&logoColor=white) ![Vercel](https://img.shields.io/badge/Vercel-000000?style=flat-square&logo=vercel&logoColor=white) ![GitHub](https://img.shields.io/badge/GitHub-181717?style=flat-square&logo=github&logoColor=white) |

**Key Contributions**

- Defined the problems faced by vacant-space owners, young creators, and local residents.
- Designed a reciprocal model in which creators use spaces at low cost while providing cultural programs for the community.
- Planned a circulation structure connecting vacant spaces, creative activities, local cultural experiences, and space-maintenance revenue.
- Designed operational assumptions including program demand, revenue, break-even point, and a six-month pilot.
- Implemented a web prototype with Kakao Maps JavaScript API.
- Deployed an accessible web prototype using GitHub and Vercel.
